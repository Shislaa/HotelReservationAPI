# HotelReservationAPI
Hotel Reservation API Demo

Prerequisite:
- Java 11
- Postgres DB

Swagger-UI for API testing: http://localhost:8080/swagger-ui.html

Features for Booking:
- Search (done)
- Update (done)
- View (done)
- Cancel (done)
- Create (done)

Application flow: Create User -> Create Hotel -> Add Rooms to Hotels -> Create Booking
